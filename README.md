# Forum BooKS - Oracle APEX Application

## 📌 Overview
Forum BooKS is a web-based application built using Oracle APEX.
The system allows managing books, shelves, readers, and reports.

## 🛠 Technologies Used
- Oracle APEX 22.2
- PL/SQL
- Oracle Database

## 📂 Project Structure
apex-project/
│
├── exports/
│   └── app_export.sql
│
├── database/
│   └── tables.sql
│
└── README.md

## 🚀 Installation Instructions

1. Import tables.sql into your Oracle schema.
2. Open Oracle APEX.
3. Go to App Builder → Import.
4. Upload app_export.sql.
5. Run supporting objects if prompted.
